Integrations

Подключения и внешние сервисы для роста канала.

Эта страница показывает доступные интеграции. Twitch уже связан с твоим backend через маршрут /auth/twitch. Для полноценного входа через Twitch нужно, чтобы в Render были заполнены TWITCH_CLIENT_ID, TWITCH_CLIENT_SECRET и TWITCH_REDIRECT_URI.

Twitch

Авторизация стримера через Twitch и получение базовых данных аккаунта.

Доступно к подключению
OAuth URL: https://streamlogic-backend.onrender.com/auth/twitch
После успешного Twitch login backend вернёт пользователя обратно на landing page.
